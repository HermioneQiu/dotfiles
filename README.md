# Fred Wu's Dot Files

## Installation

    git clone git://github.com/fredwu/dotfiles ~/.dotfiles
    cd ~/.dotfiles
    ./install

## Custom Configuration

- Copy `zsh/custom.example` to `zsh/custom`
- Make changes to `zsh/custom`
- Use the command `sr` to reload the `.zshrc` source
